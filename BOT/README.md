# 📚 Student Tracker Bot

Полнофункциональный Telegram-бот для управления напоминаниями, задачами, событиями и календарём студентов.

## 🎯 Основные функции

### 📝 Управление задачами
- ✅ Создание и редактирование задач
- 🔴 Система приоритетов (высокий, средний, низкий)
- 📊 Отслеживание статуса (TODO, В процессе, Завершено, Отменено)
- 📅 Установка сроков выполнения
- 📈 Статистика по задачам

### 🔔 Напоминания
- 📌 Создание напоминаний на определённое время
- 🕐 Автоматическая отправка уведомлений
- 🔄 Включение/отключение напоминаний
- 📋 Управление списком напоминаний

### 📅 Календарь и события
- 📆 Добавление событий в календарь
- 🔗 Синхронизация с Google Calendar
- 🏷️ Категоризация (Факультет, Личное, Экзамен)
- 📍 Добавление места проведения
- 🗓️ Просмотр событий на неделю/сегодня

### 👥 Администрирование
- 🔑 Многоуровневая система ролей (STUDENT, ADMIN, SUPERADMIN)
- 📢 Массовая рассылка сообщений
- 👤 Управление пользователями
- 📊 Статистика по пользователям
- ⚙️ Системная информация

### 📊 Статистика
- 📈 Личная статистика пользователя
- 💯 Процент выполнения задач
- 🔔 Статистика по напоминаниям и событиям
- 👥 Общая статистика системы (для админов)

## 🛠️ Технический стек

- **Python 3.11+**
- **python-telegram-bot 20.7** - работа с Telegram API
- **SQLAlchemy 2.0** - ORM для работы с БД
- **APScheduler 3.10** - планирование задач (напоминания)
- **Google Calendar API** - синхронизация с Google Calendar
- **PostgreSQL / SQLite** - хранение данных

## 📋 Требования

### Обязательные
- Python 3.11 или выше
- Telegram Bot Token (получить у [@BotFather](https://t.me/botfather))
- PostgreSQL (или SQLite для разработки)

### Опционально
- Google Calendar API credentials
- Redis (для кеширования)
